# Resume Judge AI

Resume Judge AI is an intelligent resume analysis system built using Machine Learning, Natural Language Processing, and Generative AI concepts.  
The system evaluates a user's resume, detects skills, predicts job category, calculates resume strength score, and provides feedback for improvement.

This project is designed as a full-stack AI-based application using Python, Flask, Scikit-learn, NLP, and modern frontend UI.

---

## Features

- Resume Upload System
- Resume Text Extraction (PDF)
- Skill Detection using NLP
- Job Category Prediction using ML Model
- Resume Score Calculation
- AI Feedback System
- Modern Frontend UI
- API Based Backend
- Dataset Training using Kaggle data
- Modular Project Structure

## How It Works

1. User uploads resume
2. Text is extracted from PDF
3. Skills are detected
4. ML model predicts category
5. Score is calculated
6. Feedback is generated
7. Result shown on frontend

## Installation

Install all libraries:


setup.bat


or


pip install -r requirements.txt


Install spacy model:


python -m spacy download en_core_web_sm


---

## Run Project


run.bat


or


python backend/app.py


Open frontend:


frontend/index.html
